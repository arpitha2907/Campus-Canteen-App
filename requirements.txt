# 📦 Dependencies Reference

> Note: Node.js projects use `package.json` instead of `requirements.txt`.
> This file is for documentation/reference only.

---

## Backend Dependencies (`backend/package.json`)

### Production Dependencies
```
express          ^4.18.0    Web server framework
mongoose         ^7.0.0     MongoDB object modeling
dotenv           ^16.0.0    Load environment variables
bcryptjs         ^2.4.3     Password hashing
jsonwebtoken     ^9.0.0     JWT authentication
cors             ^2.8.5     Cross-origin resource sharing
socket.io        ^4.6.0     Real-time bidirectional events
groq-sdk         ^0.3.0     Groq AI API client
```

### Dev Dependencies
```
nodemon          ^3.0.0     Auto-restart on file changes
```

### Install Command
```bash
npm install express mongoose dotenv bcryptjs jsonwebtoken cors socket.io groq-sdk
npm install --save-dev nodemon
```
